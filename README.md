## Spectral Modeling of EEG Signals

### EEG Dataset
The EEG.mat file includes:
* dalt: delta band (5000×1)
* thalt: theta band (5000×1)
* alpha: alpha band (5000×1)
* belta: beta band (5000×1)
* x12120x2Eedf: raw EEG/EMG (95000×32)

### Usage
1. Open the EEG_Spectral_Analysis.m script in MATLAB.
2. Load EEG signals.
3. Run the script to see the spectrum plots and peak detection.
